    # 在APPLoader中安装测试的app

    1. 打开模拟器 系统应用 - 文件管理器 - mnt - sdcard - UZmap - wgt - 新建文件夹
    2. 文件夹命名为APP-id，后面把配置文件复制进去，就会自动生成app。
    3. 设置共享文件夹，才可以在模拟器进行复制黏贴，模拟器顶部点击“设置”-“共享文件夹”
    4. 文件管理器找到刚才共享的工程文件，复制xml，在到刚才新建的APPid文件夹中黏贴。
      共享目录为 mnt / sdcard / droid4xshare
    5. config.xml文件中配置
    <widget id="A6007813127779"  version="0.0.1">
<content src="http://192.168.50.28:8080/platform/index.html" />

    content src 本地服务填进去，这样本地项目开发的时候，模拟器中会自动更新。
